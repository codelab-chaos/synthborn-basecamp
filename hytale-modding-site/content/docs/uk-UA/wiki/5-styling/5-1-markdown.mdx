## Форматування тексту

```md
**Жирний текст**
*Текст курсивом*
~~Закреслений~~
**_Жирний та курсивний_**
```

## Заголовки

```md
# Заголовок 1
## Заголовок 2
### Заголовок 3
#### Заголовок 4
```

## Списки

### Невпорядковані

```md
- Елемент
- Елемент
  - Вкладений елемент
  - Вкладений елемент
- Елемент
```

### Впорядковані

```md
1. Перший
2. Другий
  1. Вкладений
  2. Вкладений
3. Третій
```

### Список завдань

```md
- [x] Виконано
- [ ] Не виконано
```

## Посилання

```md
[Текст посилання](https://example.com)
[Посилання із заголовком](https://example.com "Заголовок при наведенні курсора")
```

## Зображення

Сайт може містити зображення із зовнішніх ресурсів.

```md
![Текст](https://example.com/image.png)
![Текст](https://example.com/image.png "Назва зображення")
```

## Код

### Лінійний

```md
Використовуйте `code` всередині тексту.
```

### Блок

````md
```js
const hello = "світ";
console.log(hello);
\```
````

Підтримувані мови програмування: `js`, `ts`, `css`, `html`, `json`, `md`, `bash` та інші.

## Цитати

```md
> Це блокова цитата.
> Вона може займати кілька рядків.
>
> І кілька абзаців.
```

## Сповіщення

Сповіщення – це стилізовані цитати для виділення важливої ​​інформації.

```md
> [!NOTE]
> Корисна інформація, яку має знати читач.

> [!TIP]
> Корисна порада для кращого виконання завдань.

> [!IMPORTANT]
> Ключова інформація, про яку має знати читач.

> [!WARNING]
> Потенційні проблеми, на які читач має звернути увагу.

> [!CAUTION]
> Дії, які можуть спричинити проблеми або втрату даних.
```

## Таблиці

```md
| Колонка A | Колонка B | Колонка C |
| --------- | --------- | --------- |
|  Комірка  |  Комірка  |  Комірка  |
|  Комірка  |  Комірка  |  Комірка  |
```

Вирівнювання:

```md
| Ліворуч  | Центр    | Праворуч |
| :------- | :------: | -------: |
| Комірка  | Комірка  | Комірка  |
```

## Горизонтальна лінія

```md
---
```

## Примітки

```md
Деякий текст із виноскою.[^1]

[^1]: Це вміст виноски.
```

## Уникання форматування

```md
\*Не курсив\*
\# Не заголовок
```
