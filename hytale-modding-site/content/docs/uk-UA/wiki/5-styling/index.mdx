## Загальна інформація

Вікі підтримує Markdown для форматування тексту, а також користувацькі розширення для більш просунутого вигляду.

У цьому розділі ви знайдете:

- [**Markdown**](./markdown): Заголовки, списки, таблиці, блоки коду, примітки тощо.
- [**Зображення**](./images): Як вставляти та налаштовувати зображення.
- [**Custom CSS**](./custom-css): Застосовуйте власні стилі, щоб ваша сторінка виділялася.
