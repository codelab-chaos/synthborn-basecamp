## Client Side
This is a really simple process, more or less, drag and drop.

1. First Navigate to your equivalent of `C:\Users\(Your User)\AppData\Roaming\Hytale\UserData`.
2. Open or Create the `mods` folder.
3. In a new Window/Tab open the folder in which the package you extracted earlier is located.
4. Navigate into the `mods` folder.
5. Select all items within the folder, copy them.
6. Navigate back to the games `mods` folder, and paste them.
7. Next time you launch the game, the Asset Packs you installed should be applied.

___

## Server Side
This is a pinch more involved but still, more or less, drag and drop.

1. First Navigate to your equivalent of `D:\Hytale Server\2026.02.19-1a311a592\Server`.
2. Open or Create the `mods` folder.
3. In a new Window/Tab open the folder in which the package you extracted earlier is located.
4. Navigate into the `mods` folder.
5. Select all items within the folder, copy them.
6. Navigate back to the servers `mods` folder, and paste them.
7. Next time you launch the server, the Asset Packs you installed should be applied.
