## Client Side
This is a really simple process, more or less, drag and drop.

1. First Navigate to your equivalent of `C:\Users\(Your User)\AppData\Roaming\Hytale\UserData`.
2. Open or Create the `earlyplugins` folder.
3. In a new Window/Tab open the folder in which the package you extracted earlier is located.
4. Navigate into the `earlyplugins` folder.
5. Select all items within the folder, copy them.
6. Navigate back to the games `earlyplugins` folder, and paste them.
7. Next time you launch the game, the Early Plugins you installed should be applied.

___

## Server Side
This is a pinch more involved but still, more or less, drag and drop.

1. If you want to prevent the early plugins warning you'll see when launching add the following argument to the Launch Args `--accept-early-plugins`.
2. First Navigate to your equivalent of `D:\Hytale Server\2026.02.19-1a311a592\Server`.
3. Open or Create the `earlyplugins` folder.
4. In a new Window/Tab open the folder in which the package you extracted earlier is located.
5. Navigate into the `earlyplugins` folder.
6. Select all items within the folder, copy them.
7. Navigate back to the servers `earlyplugins` folder, and paste them.
8. Next time you launch the server, the Early Plugins you installed should be applied.
