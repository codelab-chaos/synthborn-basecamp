Hytale è un gioco RPG sandbox basato su voxel di **Hypixel Studios**, come descritto sul [sito web di Hytale](https://hytale.com/):

> Parti per un'avventura pensata sia per la creatività che per il divertimento. Hytale unisce la libertà di un sandbox allo spirito di un RPG: esplora un mondo generato proceduralmente pieno di dungeon, segreti e una varietà di creature, e poi modellalo blocco dopo blocco.

Hytale è stato progettato fin dall'inizio pensando alla possibilità di modificare il gioco e ai contenuti generati dagli utenti.
The goal of **Hytale Modding** is to provide an excellent resource so that
anyone can mod Hytale themselves, regardless of skill level or specialization.

It is important to note that modding Hytale is not restricted to programming plugins only.
Programming, visual scripting, texturing, 3D modelling, modifying assets, etc., are all modding.

<u>In Hytale, chiunque può essere un modder.</u>
