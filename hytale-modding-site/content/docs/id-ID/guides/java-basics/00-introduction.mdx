Selamat datang dalam perjalananmu belajar Pemrograman Java! Panduan ini dirancang untuk mengenalkanmu pada dasar-dasar Java.

## Apa itu Java?

Java adalah bahasa pemrograman berorientasi objek yang kuat dan dibuat pada tahun 1995. Ini adalah salah satu bahasa pemrograman paling populer di dunia dan digunakan untuk segalanya, mulai dari aplikasi seluler hingga perangkat lunak perusahaan. Most importantly for us, it's the language Hytale uses for its servers & modding API.

## Apa yang akan kamu pelajari

Panduan ini disusun untuk membawamu langkah demi langkah melalui:

1. Sintaks dan Dasar-Dasar Java
2. Alur Kontrol
3. Pemrograman Berorientasi Objek

Kemudian kita hanya akan fokus pada konsep-konsep khusus Hytale. (akan ditentukan)

## Menyiapkan Lingkungan Kerjamu

Sebelum kita terjun ke pemrograman, kamu memerlukan beberapa hal penting, kamu perlu menginstal:

### Java Development Kit (JDK)

Download and install JDK 25 or above from the [official Oracle website](https://www.oracle.com/java/technologies/javase-jdk25-downloads.html) or use an open-source alternative like [AdoptOpenJDK](https://adoptopenjdk.net/).

### Integrated Development Environment (IDE)

IDE adalah aplikasi perangkat lunak yang menyediakan fasilitas lengkap bagi programmer untuk pengembangan perangkat lunak. Pilihan populer untuk Java meliputi:

* [IntelliJ IDEA](https://www.jetbrains.com/idea/)
* [Eclipse](https://www.eclipse.org/)

## Pola Pikir Modding

Modding adalah pemecahan masalah yang kreatif. Kamu akan:

* Menambahkan blok, item, dan makhluk baru ke Hytale
* Mengubah mekanika game untuk menciptakan pengalaman unik
