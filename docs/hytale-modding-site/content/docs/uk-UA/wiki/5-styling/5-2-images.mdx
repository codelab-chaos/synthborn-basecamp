## Завантаження зображень

Щоб завантажити зображення, перейдіть на сторінку керування вашою модифікацією і натисніть **Upload Files**.
Після завантаження скопіюйте посилання на файл та використовуйте його на своїй сторінці:

```md
![Текст](https://wiki.hytalemodding.dev/storage/mods/{mod-id}/files/{file-id}.png)
```

Ви також можете додати заголовок, який з’являється при наведенні курсора:

```md
![Текст](https://wiki.hytalemodding.dev/storage/mods/{mod-id}/files/{file-id}.png "Назва зображення")
```
