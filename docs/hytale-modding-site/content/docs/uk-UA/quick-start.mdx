Ласкаво просимо до посібника з модифікування Hytale! Тут ви знайдете все, що вам потрібно знати про створення власних модифікацій для Hytale.

<Callout type="info">
Hytale Modding — це, перш за все, документація, тому ми не надаємо повного посібника з модифікування.
</Callout>

## З чого почати

Якщо ви нічого не знаєте про модифікування Hytale чи навіть Java, або ж хочете освіжити свої знання, ось кілька посилань, які вам у цьому допоможуть.

* [Основи Java](./guides/java-basics/00-introduction) – вивчіть синтаксис Java та мовні функції, необхідні для модифікування.
* [Навчитися вчитися](./guides/learning-to-learn) – чому ми використовуємо Java.
* [Налаштування середовища розробки](./guides/plugin/setting-up-env) – налаштуйте свій проєкт для розробки.
* [Збірка та тестування](.guide/plugin/build-and-test) – зберіть свій проєкт.
* [Реєстрація](./guides/plugin/logging) – знайдіть та інтерпретуйте вивід вашої модифікації під час запуску гри.
* [Створення блоку](./guides/plugin/creating-block) – створіть простий блок, щоб перевірити, чи працює ваша конфігурація.

Якщо у вас є досвід або ви хочете спробувати щось складніше:

* [Генерація світу](./guides/plugin/world-gen) – додайте нового ландшафту, біомів та власних місць руди.
* [Налаштування керування камерою](./guides/plugin/customizing-camera-controls) – змінюйте поведінку камери гравця під час гри.
* [Система компонентів сутностей (ECS)](./guides/ecs/entity-component-system) – як використовувати ECS.
* [Префаби](./guides/prefabs) – створювати структури та шаблони для повторного використання для редагування світу.
* [Публікація вашого мода](./publishing) – упакуйте та поділіться своєю модифікацією з іншими.
