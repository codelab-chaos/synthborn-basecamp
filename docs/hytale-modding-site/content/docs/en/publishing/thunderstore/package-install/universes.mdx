## Client Side
This is a little involved, but this should be intuitive

1. Navigate to your equivalent of `C:\Users\(Your User)\AppData\Roaming\Hytale\UserData`.
2. Navigate into `Saves`, and enter the save you wish to put the Universe into.
3. Backup the `universe` folder.
4. In a new Window/Tab navigate to the packages `universes` folder.
5. Choose the universe you want to go with (if there are multiple), and ensure its called `universe`, copy that folder.
6. Go to the Games `Saves` folder, and paste the folder.
7. Next time you launch the world, it will now have that Universe in affect.

___

## Server Side
This is a little involved, but this should be intuitive

1. First Navigate to your equivalent of `D:\Hytale Server\2026.02.19-1a311a592\Server`.
2. Make a backup of the `universe` folder.
3. In a new Window/Tab navigate to the packages `universes` folder.
4. Choose the universe you want to go with (if there are multiple), and ensure its called `universe`, copy that folder.
5. Go to the servers root, and paste in that folder.
6. When you launch the server, the universe should take affect.
