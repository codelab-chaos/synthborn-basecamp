First things first let's get the basics out of the way.
1. Open the folder in which you downloaded your package from.
2. Extract the package into a new folder.
3. Follow the below guides on how to install the package, or follow the instructions provided in the packages `README.md`.

So you wish to install a Thunderstore Package? Well good news its really easy! Just follow the guides linked below:
