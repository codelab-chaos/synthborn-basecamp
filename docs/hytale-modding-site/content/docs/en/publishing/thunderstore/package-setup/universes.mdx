This will be a step by step guide on how to insert your universes into the thunderstore package.

1. First off navigate to your package, and create a folder called `universes`
2. In a new window/tab navigate to your equivalent of `C:\Users\(Your Username)\AppData\Roaming\Hytale\UserData\Saves`.
3. Navigate into the Save that the universe resides within.
4. Copy the `universe` folder into the `universes` folder we made earlier, rename it to whatever, for the case you want to bundle multiple universes, otherwise leave it with the default name.
5. And thats all you really need to do.
