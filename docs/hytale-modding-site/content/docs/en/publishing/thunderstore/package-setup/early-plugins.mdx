This will be a step by step guide on how to insert your early plugins into the thunderstore package.

1. Navigate to where your early plugin is, keep that folder open or memorized.
2. Within the Thunderstore Package setup we created in the prior step, create a folder called `earlyplugins`.
3. Place the early plugin (should end in `.jar`) into this folder.
4. That's pretty much all there is to it.
