This will be a step by step guide on how to insert your asset packs into the thunderstore package.

1. Navigate to where your asset pack is, keep that folder open or memorized.
2. Within the Thunderstore Package setup we created in the prior step, create a folder called `mods`.
3. Place the asset pack (should end in `.zip`) into this folder.
4. That's pretty much all there is to it.
