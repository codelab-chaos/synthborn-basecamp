# Vanilla Worldgen V2 Examples

This folder is a symlink mirror into the real vanilla assets under:

```text
../../_Assets/Server/HytaleGenerator
```

It exists so worldgen research can traverse the key examples from one place
without copying JSON files and creating stale duplicates.

Contents:

```text
Biomes/Root/          Basic and Test_Features
Biomes/Examples/      Explicit tutorial/example biomes
Biomes/Experimental/  Advanced experimental biomes
Biomes/Generative/    Procedural landmark/shape examples
WorldStructures/      Vanilla world structure examples
```

Do not edit these links directly when experimenting. Copy an asset into a test
mod first, then edit the copy.

The companion map is:

```text
../worldgen-v2-vanilla-examples-map.md
```
