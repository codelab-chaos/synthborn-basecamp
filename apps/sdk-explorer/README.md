# SDK Explorer

Static React app for wildcard searching generated Hytale Server SDK markdown
cards.

## Build

```bash
npm run build
```

This runs:

1. `npm run sync-data` — builds `data/sdk-reference.json` from `docs/sdk/*.md`.
2. `npm run build-web` — writes static assets and `index.html` in this folder.

## Data Source

The app data is generated by:

```bash
node ../../tools/refs/sdk/build-sdk-app-data.js
```

The markdown under `docs/sdk/` remains the source of truth. Do not hand-edit
`data/sdk-reference.json`; regenerate it after SDK markdown changes.
